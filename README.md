# 🔒 FocusLock

**NFC 태그로 스마트폰 앱을 차단하는 자기통제 도구**

유튜브를 열려고 하면 차단 화면이 뜨고, 공부 장소에 두었던 NFC 태그를 갖다 대야만 30분 사용할 수 있어요.
"해야 한다는 걸 알면서도 안 하는" 문제를 물리적 마찰로 해결합니다.

---

## 기능

- **NFC 태그 기반 잠금** — 등록된 NFC 태그를 갖다 대야만 차단 해제 (30분 세션)
- **다중 태그 지원** — 태그마다 다른 앱 묶음 차단 (공부방 태그, 카페 태그 등)
- **셧다운** — 외출 시 태그 갖다 대서 차단 즉시 해제
- **분할화면 / 팝업뷰 차단** — OneUI 멀티태스킹 환경도 감지
- **재부팅 후 자동 유지** — 꺼도 다시 켜지면 차단 계속 작동
- **백업/복구** — 데이터 손상 시 자동 복구

---

## 요구사항

- Android 8.0 (API 26) 이상
- NFC 지원 기기
- NFC 태그 (NTAG213 등 일반 NFC 스티커 태그)

## 설치 방법

플레이스토어 미등록 앱입니다. 직접 빌드해서 설치해야 해요.

### 1. 프로젝트 클론

```bash
git clone https://github.com/YOUR_USERNAME/FocusLock.git
```

### 2. Android Studio에서 열기

```
File → Open → FocusLock 폴더 선택
```

### 3. 빌드 및 설치

```
Build → Build Bundle(s) / APK(s) → Build APK(s)
```

생성된 `app-debug.apk`를 폰에 설치하세요.
설치 시 **"출처를 알 수 없는 앱 허용"** 필요합니다.

---

## 사용 방법

1. **PIN 설정** — 앱 설정 진입 보호용
2. **태그 추가** → NFC 태그 갖다 대기 → 이름 입력 → 차단할 앱 선택
3. **접근성 서비스 활성화** — 설정에서 FocusLock 켜기
4. **태그를 공부 장소에 배치** — 서랍 안, 특정 장소 등
5. 차단된 앱을 열면 NFC 태그를 갖다 대야 해제됩니다

---

## 주의사항

- 앱 수정/삭제 시 해당 태그로 NFC 인증이 필요합니다
- 태그를 분실하면 새 태그를 등록해서 복구할 수 있습니다 (마스터 태그 별도 보관 권장)
- 하드웨어 버튼 리커버리 초기화(볼륨+전원)는 막을 수 없습니다
